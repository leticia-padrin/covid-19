Nós somos um site sobre os acontecimentos do Covid-19, um site especializado em pesquisas e atualizações de dados sobre os acontecimentos da Covid-19 no mundo, e nossa especialidade é preparar nossos clientes para os desafios do futuro, em um cenário global e dinâmico, atráves de mapas e gráficos sobre a Covid-19, que são constantemente atualizados pelo "Brazil Covid-19 API". Api é uma interface de programação de aplicações, no contexto a palavra Aplicação refere-se a qualquer software com uma função distinta. A interface pode ser pensada como um contrato de serviço entre duas aplicações. E com essas explicações podemos manifestar nosso apredizado em aula com HTML, CSS e a utilização do JavaScript.
